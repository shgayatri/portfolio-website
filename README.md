# Computer Science Student Portfolio

A modern, responsive portfolio website for a computer science student showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Skills Section**: Highlights key programming languages, databases, and tools
- **Projects Showcase**: Displays featured projects with descriptions
- **Contact Form**: Allows visitors to send messages
- **Smooth Scrolling**: Navigation with smooth scroll to sections

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome Icons

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content to match your information

## Customization

- **Personal Information**: Update the hero section, about section, and contact details
- **Skills**: Modify the skills cards in the skills section
- **Projects**: Add or edit project cards with your own projects
- **Styling**: Adjust colors and styles in `style.css`
- **Contact Form**: For a functional form, integrate with a service like Formspree or Netlify Forms

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
